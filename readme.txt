How to run and complie JavaBankAccount 

1. Extract the contents of the zip file to a directory of your choice
2. Once the contents are extracted, move the "Data" folder to "C:\"
3. Next, navigate to ~\Project Part 3\BankAccount\target in the command prompt 
4. Next, type "java -jar BankAccount-1.0-SNAPSHOT-jar-with-dependencies.jar" to execute the program
